## Methodology

Every write-up follows this structure:

1. **Reconnaissance** — Nmap scan, service enumeration
2. **Enumeration** — Deep dive into discovered services
3. **Vulnerability Analysis** — CVE research, attack vector
4. **Exploitation** — Step-by-step attack execution
5. **Post-Exploitation** — Flags, pivoting
6. **Privilege Escalation** — Path to root
7. **Lessons Learned** — Offensive and defensive perspective
